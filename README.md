# Fullstack Task 3 — Advanced CSS & Responsive Design  
### Internship Project — Cognify Technologies  

This project demonstrates a fully styled, responsive webpage using:

- HTML5  
- CSS3  
- Bootstrap 5  
- Advanced CSS transitions & hover effects  

The webpage is designed for a **Vintage Blooms** flower shop with multiple sections and enhanced UI animations.

---

## How to Run

1. Download or clone the repository  
2. Make sure `index.html` and `style.css` are in the same folder  
3. Open `index.html` in any web browser  
4. No server setup required

---

## Features

- **Multiple Sections Included**
  - Header with navigation  
  - About Us  
  - Our Blooms (with responsive table)  
  - Florals section with images and lists  
  - Footer with contact information  

- **Responsive Design**
  - Built using Bootstrap 5  
  - Automatically adapts to mobile, tablet, and desktop  

- **Advanced CSS Styling**
  - Hover animations  
  - Smooth transitions  
  - Scale effects  
  - Shadow & opacity animations  

- **Clean UI**
  - Pastel green theme  
  - Centered floral images  
  - Elegant quotes for visual appeal
